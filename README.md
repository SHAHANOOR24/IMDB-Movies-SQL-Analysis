# IMDB Movies SQL Analysis

## Project Overview

This capstone project focuses on analyzing IMDB movie and director datasets using SQL to derive business insights and strengthen advanced query skills.

The analysis includes movie popularity, revenue performance, ratings, and director-based insights.

## Tools & Technologies

* SQL
* MySQL
* Relational Database Analysis

## Problem Statement

* Retrieve complete movie and director datasets
* Identify top-performing movies
* Analyze director performance
* Find most bankable directors
* Perform advanced joins and ranking queries

## Key Analysis Performed

* Movie count analysis
* Top 3 most popular movies
* Highest revenue-generating movies
* Top-rated movies after 2000
* Director-wise movie contributions
* Most bankable directors

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* JOIN
* COUNT
* LIMIT
* Aggregations
